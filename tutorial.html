loops

There are 3 types of loops in Nx.

They are:

  • while loop
  • do-while loop
  • for loop

While loop

Example:

print hello

endloop

If the condition that is written after the while is false then the code inside it will run if the condition is true the loop will end.

To end the loop write endloop.

do-while

Unlike for and while loops, which test the loop condition at the top of the loop, the do...while loop in Nx checks its condition at the bottom of the loop. A do...while loop is similar to a while loop, except the fact that it is guaranteed to execute at least one time.

Example:

do

print 1

for loop

there are many types of for loops in Nx.

  • for.list which is equivalent to for /l in batch.
  • file.scan which is equivalent to for /r in batch.

If statement

An if statement is a programming conditional statement that, if proved true, performs a function or displays information.

Example:

if %b%==1 print the value of b is 1.

if %b%==1 (

print the value of b is 1.

)

You can write more than one line of code that should be executed if the condition is true by ().

All the comparison operators:

"==" - Equal

"EQU" - Equal/numeric equal

"NEQ" - Not equal

"LSS" - Less than

"LEQ" - Less than or equal

"GTE" - Greater than

"GEQ" - Greater than or equal

Other kinds of if statements

ifExist

The ifExist statement checks if a file exists

Example:

ifExist b.txt print b.txt exists

The above line of code will check if a file named b.txt exists and if it exists it will print b.txt exits.

ifDefined

The ifDefined statement checks if a variable is defined.

Example:

ifDefined b print b is defined

The above line of code will check if a variable named b is defined and if its defined then it will print b is defined.

ifNot

The ifNot statement will check if the condition is not true and if it is not true it will execute the code

Example:

ifNot b==12 (

print the value of b is not equal to 12

)

The above line of code will check if the value of a variable named b is equal to 12 and then if it is nbot equal to 12 then it will print the value of b is not equal to 12

else

In programming languages, an else statement is an alternative statement that is executed if the result of a previous test condition evaluates to false.

Example:

if %b%==1 (

print the value of b is 1

) else (

print the value of b is not 1

)

else if

A elseif statement is a conditional statement that is like a if statement but runs if the previous if statement is false and it is true.

Example:

if %b%==1 (

print the value of b is 1.

) elseif %b%==2 (

print the value of b is 2

)

File IO

Write in a file

To write in a file you have to write print then whatever you want to store in that file and then >> and then the file name with file extension.

Example:

print hello >> b.txt

The above line of code will make a file named b.txt and will write hello inside it.

Read from a file

To read text from a file you have to file.read then the name of the file.

Example:

file.read b.txt

The above line of code will read the text of b.txt which we have made earlier and it will print the text inside b.txt

For the user to only press some keys you have to write key and then the keys that you want your user to be able to press.

With the above line of code the user would be only allowed to press w,s,a or d.

If you want to check which key he pressed you write ifErrorlevel== and then the key and then the code to be executed if the statement is true.

Header files

Header files in nx unlike C/C++ have the same file extension.

You store code in multiple files by creating header files. Header files are nothing but normal Nx files when we include them to our main file they excutes all the code written inside it.

Example:

print hello

We have made header file named h.nx and written print hello inside when we will include h.nx it will print hello

Include a header file

to include a header file youn have to write #include and then the name of file but with no file extension.

Example:

#include h

The above line of code will include the header file h.nx which we have earlier remember dont write the file extension while including it. It will print hello world as we have written print hello inside h.nx.
